# COVID-19 Analysis Project

## 📊 Overview
This project analyzes COVID-19 patient data to understand infection trends and recovery patterns.

## 🧰 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## ▶️ How to Run
pip install -r requirements.txt  
python src/analysis.py

## 📈 Output
- Graphs saved in outputs folder
- Recovery prediction model"# covid-analysis-project"
